# Voice-Email-Assistant
A Python-based voice-controlled email assistant that allows users to send, read, and reply to emails using speech commands. The system enhances accessibility and includes PIN-based security along with an activity dashboard for monitoring.

# Features
- 🎙️ Voice-based email interaction  
- 📤 Send emails using speech  
- 📥 Read latest emails  
- 🔁 Reply to emails  
- 🔐 PIN-based security verification  
- 📊 Activity logging using JSON  
- 📈 Dashboard visualization using Jupyter Notebook

# Technologies Used
- Python  
- SpeechRecognition  
- pyttsx3 (Text-to-Speech)  
- smtplib (Email sending)  
- imaplib (Email reading)  
- pandas & matplotlib (Dashboard)

# Security Feature
- Voice-based **PIN verification** before sending/replying emails  
- Prevents unauthorized access

# Objective
To provide an accessible and efficient email management system using voice commands, reducing dependency on traditional input devices.

# Future Scope
- Voice biometric authentication  
- GUI-based interface  
- Integration with mobile apps  
- Multi-language support
